# 讯飞小助手

一个 Chrome/Edge 浏览器扩展，用于移除讯飞同传页面的无用元素，并页面杂乱元素并最大化内容区域，专为字幕查看体验而设计。

## 功能特性

- **清爽模式**：移除不必要的 UI 元素（页头、会议标题、页脚）以最大化内容区域
- **字幕字体控制**：可调节字幕字体大小（18-100px）
- **自动滚动**：自动滚动以显示最新的字幕内容
- **持久化设置**：记住您的偏好设置，跨会话保持

## 安装方法

### 从源码安装

1. 克隆或下载此仓库
2. 打开 Chrome/Edge 浏览器，访问 `chrome://extensions/`（或 `edge://extensions/`）
3. 在右上角启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择扩展所在的目录

## 使用说明

1. 点击浏览器工具栏中的扩展图标
2. 切换开关以启用/禁用清爽模式
3. 使用滑块调整字幕大小（18-100px）
4. 设置会自动保存并应用

### 清爽模式的作用

启用后，扩展会：
- 移除 `.header`、`.meeting-title` 和 `.footer` 元素
- 移除 `.page-content` 的内边距并最大化其高度
- 在字幕容器中启用滚动
- 当新字幕出现时自动滚动显示
- 应用自定义字体大小到字幕

## 技术细节

- **Manifest 版本**：3
- **权限**：`storage`、`scripting`、`activeTab`
- **内容脚本**：在所有 URL 上运行，时机为 `document_idle`
- **使用 MutationObserver**：监控 DOM 变化，在单页应用中保持清爽模式

## 开发说明

扩展使用了：
- Chrome Storage API 用于持久化设置
- MutationObserver 用于动态内容监控
- CSS 注入用于样式覆盖
